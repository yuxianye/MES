# 制造执行系统MES

## 系统介绍

1、采用前后端分离方式

2、后端采用Asp.Net Web Api ，数据库使用SQL Server，entityformwork，工作单元仓储模式

3、桌面端采用WPF开发，使用的MVVM模式，自定义控件和界面样式
